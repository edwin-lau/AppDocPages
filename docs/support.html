常见问题

如何导入视频？

有三种方式：

  • 从相册导入：文件页右上角 + → 从相册
  • 从文件导入：文件页右上角 + → 从文件（支持 MP4、MOV 等）
  • Wi-Fi 传输：文件页右上角 + → 网络导入，在同一 Wi-Fi 下用浏览器上传

如何导入字幕？

支持 SRT 和 SRTX 格式。在文件页找到对应视频，点击字幕标签上传；也可以在播放页顶部菜单导入。将字幕文件与视频同名放在同一目录下，开启「自动加载字幕」后会自动匹配。

如何查词？

在播放页的字幕列表中，点击英文单词即可弹出词典，支持发音和加入生词本。

如何取消 VIP 订阅？

订阅管理由 Apple 负责。前往 iPhone「设置」→「Apple ID」→「订阅」中管理或取消。

数据会丢失吗？

所有学习数据（生词本、学习时长、播放进度）都存储在您的设备本地。卸载应用会清除这些数据。留言板内容通过 iCloud (CloudKit) 同步，卸载重装后留言不受影响。
